互联网广告术语
2020-09-08 # 数据指标 # 广告 # 数据指标 # 广告
  • CPM：mile，按照曝光计费
  • CPC：click，按照点击计费
  • CPA：action，按照行为计费（包含表单提交/电话拨打等行为事件）
  • **CPS：sale，按照成交计费（购买、付费）
  • CPL：lead，按照广告点击引导用户达到服务商指定网页的潜在客户数量计费
  • CPD：download，按照下载计费
  • CPR：response，按照用户的回应数量计费
  • OCPC：是CPC和CPA的升级版，OCPC以转化作为优化目标，而计费方式依旧使用CPC的点击付费，广告平台就是以点击来作为优化目标

场景举例法：
一个美好的周末，你出门逛街：到了商业街，你想知道有哪些适合你消费的店，你开始了网上搜索，叫做精准广告投放（平台、时间、人群）;
你看到了各种服装品牌的广告海报，叫做CPM（曝光）;
你看到了一个品牌，想了解具体的服装风格，详细的阅读了内容，叫做CPC（点击）;
你仔细看了网上的海报，觉得这个品牌应该很适合你，按照地址到店里逛逛，叫做CPA（行为）;
到了店里以后，无论是试穿效果还是店员的介绍都很合你心意，叫做CPS（成交）;
你开心发现了这个新品牌时，第一个想到要跟你的闺蜜分享，并把这个牌子推荐给了她，叫做CPL（潜在意向）;
买了新衣服，当然要拍美美的照片发个朋友圈，晒一晒自己的生活日记啦，叫做CPD（下载）;
在购买并结账后，你心情愉悦，并且填写了这个品牌的调查问卷，打了满分，叫做CPR（回应）
